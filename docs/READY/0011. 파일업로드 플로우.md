필요한 API 체크리스트
API	용도	상태
POST /files/upload	소용량 파일 업로드	✅ 구현됨

POST /files/multipart/initiate	Multipart 세션 시작 + DB 등록	✅ 구현됨 (fileId 반환)
POST /files/multipart/upload-part	파트 업로드 (프록시)	✅ 구현됨
POST /files/multipart/complete	완료 + NAS 동기화 큐 등록	✅ 구현됨


GET /files/multipart/parts	이어받기용 파트 목록 조회	✅ 구현됨
GET /files/multipart/uploads	uploadId 분실 시 복구	✅ 구현됨
DELETE /files/multipart/abort	업로드 취소	✅ 구현됨
